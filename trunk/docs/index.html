Welcome to Lazarus

Lazarus is a Rapid Application Development tool for Free Pascal and currently runs on Linux, Mac OS X, BSD and of course Windows.
It is freely available, open source and completely written in Free Pascal.

The official Lazarus site is http://www.lazarus.freepascal.org/.
There is a wiki and a lot of information around Lazarus at http://lazarus-ccr.sourceforge.net/.
Free Pascal can be found at http://www.freepascal.org/.
